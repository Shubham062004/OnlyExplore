# **App Name**: WanderChat

## Core Features:

- Interactive Trip Planner: Chat-based AI interface to gather user preferences for travel planning, acting as an interactive tool that determines which information should influence itinerary suggestions.
- Itinerary Visualization: Real-time itinerary display that's easy to view on multiple device types.
- Itinerary Export: Implement ability to export itineraries as shareable links, PDFs, or to calendar applications

## Style Guidelines:

- Primary color: Soft Blue (#79BAFF) for a calm and inviting travel planning experience.
- Background color: Very light desaturated blue (#F0F8FF) to keep a focus on itinerary options.
- Accent color: Warm orange (#FF9933) to highlight important CTAs and interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look with approachability, working well for both headlines and body text.
- Use outline-style icons related to travel, destinations, and activities.
- Subtle transitions and loading animations to create a smooth user experience.
